# 🤖 Machine Learning

*Machine Learning* (ML) adalah cara membuat komputer **belajar dari contoh**, bukan dari aturan yang kita tulis satu per satu. Daripada memberi tahu komputer "ini kucing karena begini-begitu", kita tunjukkan ribuan foto kucing dan biarkan ia menemukan polanya sendiri.

:::note Status
Materi ini sedang disusun. Sebaiknya selesaikan dulu jalur **[Python untuk AI](/docs/python/setup/pengenalan_python)** sebagai fondasi.
:::

## 🗺️ Rencana Materi

- **Apa itu Machine Learning** — supervised vs unsupervised, dengan analogi sederhana.
- **Alur Kerja ML** — dari data mentah sampai model yang bisa memprediksi.
- **Algoritma Dasar** — regresi, klasifikasi, dan clustering.
- **Evaluasi Model** — bagaimana tahu model kita "pintar" atau cuma menghafal.
- **Mini Project** — melatih model pertamamu dengan scikit-learn.

## 🎯 Prasyarat
- Dasar Python (jalur Python untuk AI)
- Sedikit Math untuk AI (opsional tapi membantu)
