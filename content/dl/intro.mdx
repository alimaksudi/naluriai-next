# 🧠 Deep Learning

*Deep Learning* adalah cabang Machine Learning yang terinspirasi dari cara kerja otak manusia, menggunakan *neural network* (jaringan saraf tiruan). Inilah teknologi di balik ChatGPT, pengenalan wajah, dan mobil otonom.

:::note Status
Materi ini sedang disusun. Deep Learning adalah materi tingkat lanjut — pastikan kamu sudah menguasai jalur **[Python untuk AI](/docs/python/setup/pengenalan_python)** dan dasar **[Machine Learning](/docs/ml/intro)** terlebih dahulu.
:::

## 🗺️ Rencana Materi

- **Neuron & Neural Network** — dari satu neuron sampai jaringan berlapis.
- **Training** — forward pass, loss, dan backpropagation dengan bahasa manusia.
- **Jenis Network** — CNN (gambar) dan RNN/Transformer (teks).
- **Framework** — pengenalan PyTorch.
- **Mini Project** — membangun model pengenal angka tulisan tangan.

## 🎯 Prasyarat
- Dasar Python
- Dasar Machine Learning
- Math untuk AI (vektor, matriks, turunan)

:::tip Kenapa kamu butuh ini?
Deep Learning adalah teknologi di balik ChatGPT, mobil tanpa sopir, dan penerjemah otomatis. Tapi ini materi paling atas di **[Peta Belajar](/peta-belajar)**, jadi pastikan Python, ML, dan Math-nya sudah mantap dulu ya.
:::
