## Tahap 1: Class Produk

```python
class Produk:
    """Mewakili satu produk di inventaris."""

    def __init__(self, nama, harga, stok):
        if harga < 0 or stok < 0:
            raise ValueError("Harga dan stok tidak boleh negatif")
        self.nama = nama
        self.harga = harga
        self.stok = stok

    def nilai_total(self):
        """Nilai persediaan produk ini (harga dikali stok)."""
        return self.harga * self.stok

    def __str__(self):
        return f"{self.nama} | Rp {self.harga:,} | stok {self.stok}"
```

## Tahap 2: Class Inventaris

```python
class Inventaris:
    """Mengelola kumpulan produk."""

    def __init__(self):
        self.produk = []

    def tambah(self, produk):
        self.produk.append(produk)

    def total_nilai(self):
        return sum(p.nilai_total() for p in self.produk)

    def stok_menipis(self, batas=5):
        return [p for p in self.produk if p.stok < batas]

    def laporan(self):
        print("=== LAPORAN INVENTARIS ===")
        for p in self.produk:
            print(p)
        print(f"Total nilai persediaan: Rp {self.total_nilai():,}")
        menipis = self.stok_menipis()
        if menipis:
            print("Perlu restok:", ", ".join(p.nama for p in menipis))
```

## Tahap 4: Simpan dan muat dari file (JSON)

```python
import json

def simpan(inventaris, nama_file):
    """Menyimpan inventaris ke file JSON."""
    data = [
        {"nama": p.nama, "harga": p.harga, "stok": p.stok}
        for p in inventaris.produk
    ]
    with open(nama_file, "w") as f:
        json.dump(data, f, indent=2)

def muat(nama_file):
    """Membuat inventaris dari file JSON."""
    inventaris = Inventaris()
    try:
        with open(nama_file) as f:
            data = json.load(f)
        for d in data:
            inventaris.tambah(Produk(d["nama"], d["harga"], d["stok"]))
    except FileNotFoundError:
        print("File belum ada, mulai dengan inventaris kosong.")
    return inventaris
```

## Spesifikasi tugasmu

Bangun aplikasi lengkap yang bisa:

1. **Menyimpan produk** memakai class `Produk` dan `Inventaris` (dengan validasi input).
2. **Menu interaktif** memakai loop `while`: tambah produk, lihat laporan, simpan, keluar.
3. **Menyimpan dan memuat** data dari file JSON, sehingga data tidak hilang saat program ditutup.
4. **Laporan analisis**: total nilai persediaan, produk yang stoknya menipis, dan produk termahal.
5. **Kode rapi**: nama jelas (PEP8), function pendek, dan docstring di tiap class/function.
